Basics of Web Development – Course Projects (Finland 🇫🇮)

This repository contains a collection of tasks and project completed during the Basics of Web Development course at Centria University of Applied Sciences in Kokkola, Finland. It demonstrates my progression from learning basic HTML tags to building a fully functional, responsive website.

🛠️ Part 1: Fundamentals (Task A – Task H)
These tasks focused on mastering the core building blocks of web development.

Task A - E: Introduction to semantic HTML5 structure, text formatting, and working with links and images. Implementing CSS3 for styling, including the Box Model, typography, and basic layouts.

Task F - H: Creating Weekly Timetable using JS, upgrading weekly.

🌐 Part 2: Website Development (Task I – Task L)
The final stage of the course involved designing and developing a comprehensive website, applying all previously learned techniques.
